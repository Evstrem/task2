# Password Manager PWA

## Описание

Это прогрессивное веб-приложение (PWA) — менеджер паролей, которое можно установить на мобильное устройство или использовать в браузере.  

Приложение хранит данные локально в браузере с помощью `localStorage` .
Пользователь может:

- Добавлять новые пароли (сайт, логин, пароль)
- Генерировать пароли с выбором длины и сложности
- Просматривать и удалять сохранённые пароли
- Использовать приложение офлайн

## Использование

### Добавление пароля

1. Введите URL сайта, логин и пароль
2. Нажмите кнопку Добавить
3. Пароль появится в списке ниже

### Генерация пароля

1. Выберите длину пароля 
2. Выберите сложность:
   - Лёгкий — только буквы  
   - Средний — буквы + цифры  
   - Сложный — буквы + цифры + спецсимволы
3. Нажмите Сгенерировать пароль
4. Пароль появится в поле "Пароль", его можно сразу добавить

### Удаление пароля
- Нажмите кнопку Удалить напротив нужного пароля в списке

- Используются иконки:
  - `favicon.ico` — для  браузера
  - `icons/icon-1.png` и `icons/icon-2.png` — для  PWA
